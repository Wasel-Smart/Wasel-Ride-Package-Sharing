# Observability Assets

These files provide the baseline deployment scaffolding for production telemetry:

- `otel-collector.yaml`: traces and metrics fan-out
- `prometheus.yml`: scrape targets and rules starter
- `loki-config.yaml`: centralized log store starter
- `grafana-dashboard-wasel-overview.json`: mobility platform dashboard seed

The intent is to make observability deployable, not just documented.
