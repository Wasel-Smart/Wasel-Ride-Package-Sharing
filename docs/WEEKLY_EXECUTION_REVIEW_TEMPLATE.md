# Weekly Execution Review Template

## Inputs
- Owner status by domain
- KPI movement versus target
- Release status
- Incident and support summary
- Commercial changes and contract health

## Review Agenda
1. Review last week commitments.
2. Mark each domain red, yellow, or green.
3. Name the single biggest execution bottleneck.
4. Confirm owner and due date for the next fix.
5. Escalate any unresolved red item older than one cycle.

## Required Outputs
- One-page red/yellow/green summary
- Next-week commitments by owner
- Escalation log
- KPI deltas with commentary
- Playbook updates if process changed
