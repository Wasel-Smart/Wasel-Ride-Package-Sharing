## Architecture Summary

Wasel is a Jordan-focused mobility and logistics platform featuring:

- Ride sharing with geospatial matching
- Package delivery with handoff tracking
- Corridor-based route discovery
- Bus booking integration
- Operator workflows

Built with React 18, TypeScript 5, Vite 6, Supabase (Postgres + PostGIS + Auth), TanStack Query v5, Stripe payments, deployed to Kubernetes with Redis Streams event bus.
