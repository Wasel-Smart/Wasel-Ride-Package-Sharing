## 🎯 Quick Reference

### Pre-Deployment Commands
```bash
npm run production:validate    # Validate environment
npm run verify                 # Full test suite
npm run build                  # Production build
```

### Deployment Commands
```bash
npm run production:deploy      # Deploy to Vercel
vercel --prod                  # Direct Vercel deploy
netlify deploy --prod          # Netlify deploy
```

### Post-Deployment Commands
```bash
npm run production:health      # Health check
npm run production:monitor     # Continuous monitoring
npm run backup:db              # Database backup
```

### Emergency Commands
```bash
vercel rollback [url]          # Rollback deployment
npm run backup:db              # Emergency backup
```
