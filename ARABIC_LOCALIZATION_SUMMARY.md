- "كراسي فاضية" instead of "مقاعد متاحة"

## 🎓 How to Use

### For Users:
1. Open the app
2. Click language selector
3. Choose "العربية"
4. Enjoy natural Jordanian Arabic!

### For Developers:
```typescript
import { useLanguage } from '@/contexts/LanguageContext';

function MyComponent() {
  const { t } = useLanguage();
  return <h1>{t('landing.hero.title')}</h1>;
}
```

### For Translators:
1. Open `src/locales/translations.ts`
2. Find the section to translate
3. Use natural Jordanian expressions
4. Follow the style guide
5. Test in context
